Resumo (alto nível)

  • Frontend: nginx_sites distribui requisições para site1, site2, site3.
  • Cada site é um HTML estático com botão "Executar" que faz uma requisição para nginx_api (http://SEU_IP:8081/).
  • nginx_api faz balanceamento round-robin entre api1, api2, api3, sem healthcheck.
